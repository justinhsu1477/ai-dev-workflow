## Your Task

Check for these common hidden bugs:
- Buttons that don't respond to clicks (onClick not bound)
- Forms that submit but don't send data (empty body)
- API paths that are wrong (404 errors in console)
- Data that doesn't update after save
- Elements that should appear but are missing
- Navigation links that go to wrong pages

If testing is complete (all critical flows verified), set action to "DONE".
